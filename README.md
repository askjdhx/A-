# A20_project
## Git常用命令
### 克隆
### git clone https://github.com/Stella050527/A20_project.git
### 把develop克隆到本地
### git clone -b develop https://github.com/Stella050527/A20_project.git
### 切换分支到develop
### git checkout develop
### 拉取（在推送之前一定要拉取）
### git pull
### 推送
### git add
### git commit -m"提交内容的描述"
### git push
### 将develop分支合并入main
### git checkout main
### git merge develop
